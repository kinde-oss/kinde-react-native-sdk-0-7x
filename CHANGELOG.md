#### [Unreleased]

##### Breaking Changes

- **Minimum iOS version is now 12.4+** - This SDK now uses `react-native-app-auth` (replacing `react-native-inappbrowser-reborn`) for OAuth authentication, which depends on AppAuth-iOS requiring iOS 12.0+. The deployment target has been set to iOS 12.4 for compatibility. If your app previously targeted iOS versions below 12.4, you will need to update your `IPHONEOS_DEPLOYMENT_TARGET` accordingly.

##### Added

- Replaced `react-native-inappbrowser-reborn` with `react-native-app-auth` for more reliable OAuth authentication
- Added example app in `example/` folder demonstrating SDK usage
